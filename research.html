GPU assisted state-space method for phase retrieval from noisy fringe patterns

M. Tech. Thesis

  • In the current work, a state-space method is proposed for phase estimation in digital holographic interferometry (DHI). The technique relies on windowed processing of fringe pattern using first order polynomial phase approximation and formulating a state vector model using Extended Kalman Filter for recovering the polynomial coefficients. The simulation results corroborate the performance of the proposed method at varying levels ofnoise. In addition, the computational efficiency of the proposed method is significantly improved by using parallel computations on the graphics processing unit (GPU), and the gains in execution speed are demonstrated for different sizes of holograms.
